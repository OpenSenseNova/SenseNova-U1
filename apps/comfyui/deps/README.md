# deps

This directory is reserved for future local inference integrations.

The current release uses the SenseNova token/API service only. Do not place model weights,
virtual environments, build outputs, or generated caches in this repository.

If a future local backend vendors a third-party project here:

- Keep the upstream project's license and notices.
- Document the upstream source, version, commit, and local changes.
- Prefer submodules or installation instructions when the dependency is large.
- Do not commit checkpoints, weights, or other large binary artifacts.
